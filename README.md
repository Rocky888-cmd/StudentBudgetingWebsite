# Student Budgeting System

A professional web-based budgeting dashboard designed for students to manage their daily allowance and expenses efficiently. This system helps students monitor spending, track daily budgets, and visualize expense patterns using interactive charts.

## Features

- **User Authentication**: Login using Student ID and Birthdate, ensuring only registered users can access the dashboard.
- **Allowance Management**: Set daily, weekly, or monthly allowance with automatic calculation of remaining budget.
- **Expense Tracking**: Add expenses with category, description, and amount.
- **Expense Ledger**: View a complete table of all transactions.
- **Visual Charts**:
  - **Bar Chart**: Shows weekly expense trends.
  - **Pie Chart**: Displays expense breakdown by category.
- **Professional Dashboard UI**:
  - Horizontal summary cards for Daily Budget, Spent Today, and Remaining.
  - Modern card design with hover effects and responsive layout.
  - Modal popups for adding allowance and expenses.
- **Responsive Design**: Fully responsive and mobile-friendly.

## Technologies Used

- HTML5, CSS3, JavaScript (ES6 Modules)
- Firebase Realtime Database for storing allowance and transactions
- Chart.js for interactive data visualization
- Modern CSS techniques (flexbox, grid, card shadows, hover effects)
